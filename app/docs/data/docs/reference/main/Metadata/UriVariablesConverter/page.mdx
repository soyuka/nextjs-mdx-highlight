### convert

Takes an array of strings representing URI variables (identifiers) and transform their values to the expected type.To handle the composite identifiers type correctly, use an `uri_variables_map` that maps uriVariables to their uriVariablesDefinition.
Indeed, a composite identifier will already be parsed, and their corresponding properties will be the parameterName and not the defined
identifiers.
